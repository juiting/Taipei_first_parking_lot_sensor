# 第一停車場 地磁即時監控 — Windows 安裝說明

> 適用：Windows 10 / 11。安裝後本程式在背景常駐，開機自動啟動；
> 平常只要點桌面捷徑就能看到停車場 3D 即時畫面。

## 一、安裝（只需做一次）

1. 將下載的 zip **解壓縮**到固定位置（建議 `C:\P1_Geomag_Monitor`，不要放桌面或暫存區）。
2. 進入資料夾，**雙擊 `install.bat`**。
3. 依畫面提示輸入地磁系統連線資訊（**網址、帳號、密碼**——請向系統負責人索取，僅首次需要輸入）。
4. 看到「安裝完成」即可。桌面會出現兩個捷徑：
   - **第一停車場監控**：一般視窗開啟（日常使用）
   - **第一停車場監控-全螢幕**：電視牆/簡報用全螢幕（按 `Alt+F4` 離開）

## 二、日常使用

- 點桌面「**第一停車場監控**」即可查看。
- 畫面說明：綠=空位、紅=在席、灰=維護、紫=感測器離線；**紅色車身=5 分鐘內剛停入**。
- 可用滑鼠拖曳旋轉、滾輪縮放；點任一車格看詳細資訊；下方按鈕可切換「立體／俯視／入口」視角。
- 程式開機自動啟動、異常自動重啟，平常不需要任何操作。

## 三、更新版本

雙擊 **`update.bat`**，會自動下載最新版並完成更新（設定與歷史資料都會保留）。

## 四、移除

雙擊 **`uninstall.bat`**（移除自動啟動與捷徑），再刪除整個資料夾即可。

## 五、常見問題

| 狀況 | 處理方式 |
|------|---------|
| 防毒軟體跳出警告 | 本程式未數位簽章，請於防毒軟體將整個資料夾加入「信任／排除清單」 |
| 畫面顯示「連線中斷」 | 檢查電腦網路；程式會自動重連，恢復後畫面自動更新 |
| 捷徑打開是空白頁 | 等約 30 秒（服務啟動中）再重新整理；仍異常請看 `logs\geomag.log` |
| 輸入錯帳密想重設 | 用記事本開啟資料夾內 `.env` 修改後，重新執行 `install.bat` |
| 畫面卡頓 | 切到「俯視」視角可大幅降低顯示負擔 |
| 想調整更新頻率 | `.env` 內 `POLL_INTERVAL=8`（秒），改完重開機或重跑 `install.bat` |

## 六、安全注意事項

- `.env` 內含地磁系統帳密，請勿外流此資料夾；建議這台電腦設定登入密碼。
- 本程式僅在本機提供服務（127.0.0.1），不對外開放網路埠。
